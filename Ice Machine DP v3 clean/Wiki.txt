Ice Machine:

The ice machine is a machine that freezes the water above it when powered. The ice machine can be powered for a maximum of 15 seconds on one redstone dust.
The ice crystal inside of the machine will display how much power is left.
It will only freeze a source block above the machine, so it is recomended to use a infinite water supply above the machine.
